Proje Özellikleri

Modern web geliştirmenin tüm avantajlarını bir arada sunuyoruz

⚡ Yüksek Performans

Next.js'in SSR, SSG ve ISR özellikleri sayesinde ultra hızlı sayfa yükleme süreleri ve kusursuz kullanıcı deneyimi.

📱 Mobil Uyumlu

Tüm cihazlara kusursuz uyum sağlayan, modern flexbox ve grid tabanlı duyarlı tasarım.

🔍 SEO Optimize

Arama motorlarında üst sıralar için optimize edilmiş, Next.js Metadata API ile güçlendirilmiş yapı.

🎨 Modern UI

Tailwind CSS ve Framer Motion ile tasarlanmış, animasyonlu, sade ve kullanıcı dostu arayüz.

🌗 Tema Desteği

Kullanıcı tercihlerine göre dinamik karanlık/aydınlık tema desteği ve sistem teması entegrasyonu.

🐳 Docker Desteği

Konteynerleştirilmiş yapı sayesinde hızlı ve kolay dağıtım imkanı. CI/CD pipeline'ları için hazır.
